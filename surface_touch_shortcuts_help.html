Surface Touch Shortcuts

A small always-on-top Windows panel for touch devices. It stays out of the typing focus path and gives you quick access to Wispr Flow and common editing keys.

Buttons

  • Wispr Hold
    Tap once to hold Ctrl+Win. Tap again to release.
  • Ctrl+Shift
    Toggles the modified arrow behavior.
  • Undo / Redo
    Sends Ctrl+Z and Ctrl+Y.
  • Cut / Copy / Paste
    Sends standard edit shortcuts.
  • Arrow Keys
    Use the icon buttons for navigation.
  • Enter / Delete
    Useful for quick text cleanup after selection.
  • Small-scale labels
    At very small sizes, some labels shorten automatically to keep the panel compact.
  • Hide
    Hides the panel to the system tray.
  • Quit
    Closes the app completely.

Configuration

The app reads touch_shortcuts_config.json from the same folder as the executable. Supported startup settings include ui_scale, button sizes, font, padding, topmost behavior, start hidden, and initial window position.

When you finish resizing the panel, the current ui_scale is saved back to the JSON file automatically.

Install Location

The packaged installer copies these files into C:\Program Files\Surface Touch Shortcuts:

  • The executable
  • touch_shortcuts_config.json
  • This help file

The installer is architecture-neutral. A package built on x64 installs the surface_touch_shortcuts_x64.exe build, and a package built on ARM64 installs the surface_touch_shortcuts_arm64.exe build.

The Start Menu folder includes both a normal launch shortcut and a Surface Touch Shortcuts (Admin) shortcut for cases where the target app is elevated.

If you rebuild the EXE, rerun the installer to refresh the installed copy.

Notes

If a target app is running elevated, use the Surface Touch Shortcuts (Admin) shortcut from the Start Menu. The Help button in the app opens this file from the installed folder.
